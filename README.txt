Chen Family Dashboard — Starter

1. Open index.html to preview the design.
2. To connect Google Calendar, get a public iCal URL:
   Google Calendar web → Settings → choose calendar → Integrate calendar → Public address in iCal format.
3. Paste the URL into script.js at CALENDAR_ICAL_URLS.
4. Later, host the folder on GitHub Pages / Netlify / Cloudflare Pages.
5. Open the hosted URL on iPad Safari and lock it with Guided Access.

Note: This starter shows demo events first. The next step is to connect real Google Calendar data.
